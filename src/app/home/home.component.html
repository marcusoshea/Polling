Welcome to the Polling Website!

If your order has been setup to use this application you can complete the following tasks:

Visitor (Not Logged In):

    Register Page:

    • Order Members can register for an account on this page
    • After registering new members will need to wait for approval from the order clerk or assistant clerk
    • You will need to register for each order you are a member of

    Login Page:

    • Login to website once your registration has been approved
    • You can reset your password here if you have forgotten it

Member (Logged In):

    Profile Page:

    • Update your name, email and password
    • Set yourself to active or inactive
      • This is in case you still want order access but do not want to be considered for purposes of pollings

    Pollings Page:

    • If there is not an active polling this page will indicate this
    • If there is a polling active, all candidates the order clerk/assistant has added will be available for your notes and vote.
    • You can save as a draft your notes/votes, these will not be viewable by the rest of the order
    • Once you submit your polling vote/note you cannot change them, but they will be viewable by all members of the order
    • You can view each candidate's notes/votes by clicking their name, this will open a small window, and will not reset any information you have entered into the form.
    • Votes available (dependent on your order) are:
      • Yes, this candidate is ready to join the order
      • Wait, this candidate is showing good progress but is not yet ready to join the order
      • No, this candidate is not ready to join the order
      • Abstain, you do not have an opinion on this candidate

    Candidates Page:

    • You can view all candidates in the system for your order and view external (notes not given for a polling) as well as notes that have been submitted for pollings.
    • You can also create an external note for a candidate after selecting their name.

    Polling Report page:

    • Create a report of which candidates should be recommended based on the most recent closed polling and participation/percentage of “Yes” votes. This page will also allow Order Clerks to download this report to PDF.

Order Clerk/Assistant (Logged In):

    Admin Page:

    • You can set the order and assistant clerks
    • You can approve new registrations (you must remove their candidate profile at this time!)
    • You can remove members (they will still be in the system just flagged to not be able to login anymore. This is a nuclear option and should be used in the case of permanent removal from the order.
    • You can deactivate members, this will make them not count towards participation percentages if required by your order
    • You can add or remove candidates
    • You can create pollings and select all or specific candidates
    • You can remove pollings


Functionality not completed yet:
